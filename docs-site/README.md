# Masqr.id - Public Documentation

This repository contains the public documentation for Masqr.id, a privacy-first Brand-to-Influencer Referral Platform.

## What is Masqr.id?

Masqr.id connects brands with influencers through a simple, privacy-first referral marketing platform. Key features include:

- **Privacy-First Attribution**: No cookies, hashed IP addresses, GDPR compliant
- **Transparent CPC Billing**: Pay only for actual clicks with real-time tracking
- **Anonymous Influencer Support**: Influencers can stay completely anonymous
- **Simple Setup**: Create campaigns in minutes, no complex integrations

## Privacy & Security

Our platform is designed from the ground up with privacy as a core principle:

- ✅ No tracking cookies
- ✅ No personal data storage (beyond payout details)
- ✅ Hashed IP addresses and User-Agent strings
- ✅ AES-256 encryption at rest
- ✅ TLS 1.3 for all traffic
- ✅ GDPR and e-Privacy compliant

See our [Privacy page](https://yourusername.github.io/masqr.id/privacy) for complete details.
